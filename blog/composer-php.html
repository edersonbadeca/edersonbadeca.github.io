Composer chega de includes

Como gerenciar as dependências do seu projeto de maneira fácil e organizada .

Junho 26, 2015

Composer é um gerenciador de dependencias para php. Permite você declarar as dependencias do seu projeto e instala-las no seu projeto.
Você precisará de 3 coisinhas para usar:

Um arquivo composer.json com as declarações de dependencias por ex: 
{
    "require": {
        "monolog/monolog": "1.2.*"
    }
}
Instalar o executável do composer: 
curl -sS https://getcomposer.org/installer | php

O instalador irá fazer alguns checks nas suas configurações do php e depois fará o download do arquivo composer.phar no diretório do seu projeto.
Instalar as dependencias: 
php composer.phar install
O composer irá criar um diretório chamado vendor dentro diretório do seu projeto com as dependencias declaradas no composer.json e Pronto!
Hora de usar:
Bem além de instalar as dependencias do seu projeto, o composer prepara um autoloader para você incluir no bootstrap (pré-processamento) do seu projeto: 
require __DIR__ . '/vendor/autoload.php';

Feito isto, você pode usar as suas dependencias sem choro :D
$log = new Monolog\Logger('name');
$log->pushHandler(new Monolog\Handler\StreamHandler('app.log', Monolog\Logger::WARNING));

$log->addWarning('Foo');
